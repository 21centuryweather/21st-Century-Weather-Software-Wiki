# Global Atmospheric Modeling

**ACCESS-AM3**

ACCESS-NRI has developed a global Atmospheric (AM) suite using the Unified Model and CABLE for global simulations.

The first alpha release of ACCESS-AM3 (n96) is now available. You can find all relevant information at the [ACCESS-Hive Docs](https://forum.access-hive.org.au/t/access-am3-release-information/5446) website maintained by ACCESS-NRI.
